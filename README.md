## Módulos

Eu divido minhas configurações como módulos

- `sistemas operacionais` ==> diversas configurações de diferentes sistemas operacionais
- `bin` ==> alguns scripts
- `softwares` ==> arquivos de texto com uma lista de softwares para instalar
- `outros` ==> basicamente outras configurações
